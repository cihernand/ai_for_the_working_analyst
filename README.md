## AI Working Analyst Portfolio

This repository includes practical Data Science projects developed as part of my courses at Colegio de Matemáticas Bourbaki.

The projects span NLP, computer vision, fraud detection, and time-series anomaly detection workflows, covering data preparation, model training, evaluation, and decision-ready insights. The notebooks emphasize model comparison and trade-offs (performance, interpretability, and computational cost).

### Projects
- Sentiment analysis with RNN for customer reviews.
- Image classification benchmark: custom CNN vs ResNet50 transfer learning.
- Retail fraud classification with supervised ML on imbalanced data.
- Time-series anomaly detection with Conv1D vs LSTM autoencoders.
